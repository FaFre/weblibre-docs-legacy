WebLibre supports multiple user profiles with complete data separation, allowing multiple people to use the same device or a single user to maintain separate browsing identities for different contexts (work, personal, testing, etc.).

## What Are User Profiles?

User profiles provide complete isolation between different browsing sessions. Each profile has its own:

- **Browsing Data** - Separate history, bookmarks, downloads, and cached data
- **Privacy Settings** - Independent privacy and security configurations
- **Extensions** - Each profile can install and configure different add-ons
- **Containers** - Separate container configurations and assignments
- **Tabs** - Isolated tab sessions and tree structures
- **Web Feeds** - Individual RSS feed subscriptions per profile
- **Search Settings** - Custom search engines and bang configurations
- **Gecko Profile** - Separate Gecko engine profile with independent preferences
